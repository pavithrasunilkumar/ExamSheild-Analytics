# SAP ERP Master Data Layer — Deferred

This folder is intentionally empty for now.

Per the project roadmap, SAP-style master data modeling (Center, Invigilator,
and Infrastructure Asset extracts styled after SAP Plant/Personnel/Asset
Master conventions) is being built as the final layer, once the Power BI
dashboards are complete — done once, deliberately, rather than reshaping
these tables twice.

**Planned contents:**
- `sap_center_master.csv` (Plant/Location-style fields)
- `sap_invigilator_master.csv` (Personnel Number-style fields)
- `sap_asset_master.csv` (Fixed Asset-style fields)
- `sap_integration_notes.md` — explicit documentation of what is modeled/simulated
  vs. any real SAP system connection
